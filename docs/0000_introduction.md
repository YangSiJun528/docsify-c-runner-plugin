# 들어가며

C언어 정리글

복습 목적의 자료와, 나중에 참고하기 좋은 치트시트, 가이드 느낌의 문서를 제공하는 것이 목표

나중에 아마 몇몇 사람들한테 공개할 수도 있는데, 유료 강의 내용이 담겨있어서 외부에 공유하지는 말아주었으면 함.

개인적인 의견이 담긴 문서로, 정확하지 않은 내용이 있을 수 있음.   
중요한 경우 공신력있는 문서를 제공하는데, 그걸 참고하가.


#### C언어 개요

- 기본적인 프로그래밍 기능 (반복, 조건문, 함수 등은 설명 안하고 넘어갈꺼임)

- C는 절차적 언어
    - COMP1000(프로그래밍 입문)강의에서처럼 C#도 절차지향형으로 짤 수도 있긴 했음
        - 프로그래밍의 패러다임은 대체가 아닌 상호 보완적인 관계
    - C는 순수하게 절차적 언어
    - 즉, C로 작성한 코드는 데이터보다 프로세스에 중점이 맞춰져 있음
    - 절차적 언어는 이해하기 쉬움
        - 컴퓨터의 동작과 비슷함
        - OOP, FP 처럼 추상적인 개념이 없음
        - 이러한 추상 개념으로 인한 의견 차이가 없음
    - 클래스 없음
    - 함수는 모두 전역변수
    - 변수는 함수 바깥에서 전역변수, 함수 내부에선 지역변수
